Privacy Policy

Usage of Data

We do not collect any User Data, we only collect data necessary for the bot to work.

The bot may use stored data as defined below for different features including but not limited to:

  • Command Handling
  • Sending Messages in Discord
No usage of data outside of aforementioned cases will happen and no data is shared with any 3rd parties.

Stored information

The bot may store the following information when being invited to a new Discord Server:

  • id the Id of the Discord Guild.
  • name the name of the Discord Guild.
  • f1-channel the id of the Discord channel where to create threads for F1 (Optional).
  • f1-role the id of the Discord Role where to notify (Optional).
  • f2-channel the id of the Discord channel where to create threads for F2 (Optional).
  • f2-role the id of the Discord Role where to notify (Optional).
  • f3-channel the id of the Discord channel where to create threads for F3 (Optional).
  • f3-role the id of the Discord Role where to notify (Optional).

The bot may store the following information when a new thread is being created:

  • guild The id of the Discord Guild.
  • id The thread id for the created thread.
  • date_created The Second-Exact timestamp of thread creation.

No other information outside of the above mentioned will be stored.

Temporarely stored information

The Bot may keep information stored in a transient, internal cache for some time.
After some time the cached information will be dropped and only be re-added when required.
